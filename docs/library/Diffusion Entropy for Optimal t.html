find_knee_point

 find_knee_point (y, x=None)

Returns the x-location of a (single) knee of curve y=f(x)

Type Default Details
y array, shape=[n] data for which to find the knee point
x NoneType None indices of the data points of y,
if these are not in order and evenly spaced
Returns int

compute_von_neumann_entropy

 compute_von_neumann_entropy (data, t_max=100)

Determines the Von Neumann entropy of data at varying matrix powers. The user should select a value of t around the “knee” of the entropy curve.

Type Default Details
data
t_max int 100 Maximum value of t to test
Returns array, shape=[t_max] The entropy of the diffusion affinities for each value of t
